# DTIUI

Projeto criado com [Angular CLI] versão 8.3.29.

## Executar o projeto FRONTEND
- Utilize o VSCode como IDE para abrir o projeto.
- No terminal integrado da ferramenta (Ctrl + Shift + ') execute o comango **npm install** para baixar as dependências do **Node**. Após a conclusão do processo, execute o comando **ng serve**. A aplicação será iniciada no endereço **http://localhost:4200/**.

**Débito Técnico**
- Devido a falta de tempo não foi possível implementar mensagens (erro, sucesso, validações) no cliente, mas as validações existem na API. Há problema de renderização dos dados no UPDATE e detalhamento das informações.
